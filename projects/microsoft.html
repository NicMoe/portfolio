Microsoft Assessment Quiz

Problem

Moving to the cloud can be a daunting task for IT professionals. There are many things to consider like current infrastructure, user needs, and management support before making the move. Microsoft was looking to create a fun tool to help IT professionals understand how ready they and their peers are for the cloud, as well as provide resources to help them get up to speed if they’re not ready.

Solution

Our team proposed an interactive quiz that would assess on a high level where the user stands in terms of cloud-readiness. Through a set of simple multiple-choice questions, the quiz would assess the user’s situation and give them a cloud ranking along with relevant resources based on their result. As an added bonus, the user can see how their answers compares to their peers.

Execution

To develop the quiz, it was important that I developed it in a modular component-focused way to allow for easy additions of new features and iterative improvements. This approach turned out to be very beneficial when we were asked to translate the quiz into about 20 different languages.

Tools

  • Backbone.js for the framework to have a simple object-oriented architecture
  • D3.js for generating charts
  • Webpack to modularize application Javascript
  • Sass for a powerful CSS structure that was easy to manage
  • And of course, good ol’ HTML
